# 🧠 NeuralRetail

### AI-Powered Retail Intelligence Platform

NeuralRetail is an end-to-end **AI-powered Retail Intelligence Platform** built using **Python, Machine Learning, Streamlit, and Plotly**. The platform transforms raw retail transaction data into actionable business insights through interactive dashboards, predictive analytics, customer intelligence, and inventory optimization.

Developed as part of a Data Science internship project, NeuralRetail demonstrates the complete machine learning workflow—from data preprocessing and feature engineering to model development and deployment.

Check out at: 
https://neural-retail.streamlit.app/

---

## 📌 Project Overview

Retail businesses generate millions of transactions every day. Extracting meaningful insights from such large datasets manually is time-consuming and inefficient.

NeuralRetail addresses this challenge by integrating data analytics, machine learning, and interactive visualization into a unified platform that enables businesses to:

- Monitor overall business performance
- Understand customer purchasing behaviour
- Identify customers at risk of churn
- Forecast future product demand
- Optimize inventory management
- Support data-driven decision making

---

## 🎯 Objectives

- Perform exploratory data analysis on retail transaction data
- Build a complete feature engineering pipeline

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shalvi-Surve/NeuralRetail.git
```

Move into the project directory

```bash
cd NeuralRetail
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch the dashboard

```bash
streamlit run app.py
```
